The Address Book project is a console-based application developed in C++ that allows users to efficiently manage and store their contacts. This program enables users to perform key operations such as adding new contacts, editing existing contact details, deleting contacts, and searching for contacts by name or phone number.

Each contact entry stores essential information, including:

Name
Phone Number
Email Address
Home Address
The project demonstrates the use of file handling in C++ for persistent storage, ensuring that contact information is saved to a file and can be retrieved or updated during future sessions. The use of object-oriented programming (OOP) principles helps manage contact data through classes and objects, making the code modular and easy to maintain.

Key Features:
Add new contacts with essential details.
Edit or delete existing contacts.
Search contacts by name or phone number.
List all contacts in alphabetical order.
Persistent storage using file I/O operations.
Simple, text-based user interface for easy interaction.
This project is a great demonstration of basic file handling, object-oriented programming, and data management techniques in C++. It is ideal for beginners learning how to work with user input, data structures (arrays or vectors), and file systems in C++.
